  2. Element, class, & id Selectors;

    • Element Selector: It is simply when we apply some sort of style to the element such as the "p" tag, which is an example of an element.

    • Class Selector: It is basically used when we define some element's(attribute's) name-value pair, or we say it as when we define an element such as "p" element's class, and directly get the ability to access the style properties of the certain class wherever it is used.

      Targets one or more elements with the same class. Multiple elements can have the same class, and a class can be applied to multiple elements. Used to group and apply shared styles or JavaScript functionality to multiple elements.

    • Id Selector: Should be unique within the HTML document. An ID cannot be used multiple times. Targets a single unique element on the page.
